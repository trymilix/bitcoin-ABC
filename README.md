# 吹吧 比特币

最早听说比特币，还是2011，那时每个比特币大概6、7美元。

再一次就是2013、2014了，这一次是被媒体轰炸的，价格也已经是1000美元左右了。

大多数人了解比特币，也就是始于这个时候吧。

接踵而来的政府打压，又让比特币归于沉寂。

每次看到比特币的报道，第一反应就是：吹吧，比特币，看你还能火多久。
