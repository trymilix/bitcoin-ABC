# 管好你的钱包

[Bitcoin Core](https://github.com/bitcoin/bitcoin) 调用 [RAND_bytes](https://www.openssl.org/docs/manmaster/crypto/RAND_bytes.html) 生成256位随机数，即比特币钥匙。

比特币钥匙容易生成但不宜保管，这就是比特币钱包需要考虑的问题。
